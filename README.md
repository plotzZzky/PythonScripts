<h2 align="center"> GitGet </h2>

![gitget](https://github.com/plotzZzky/PythonScripts/assets/12895974/f9ece8f3-3275-4d3f-b08e-8fb620fdbb46)

### Caracteristicas:

- [x] Busca por username;
